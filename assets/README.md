# Branding assets (Emmanuel Catholic College)

Place your official assets here so the game uses college branding:

- **logo.svg** or **logo.png** — ECC logo (used in the app header). Recommended height ~40px.
- **employability/** — Optional: replace the built-in skill icons with your Employability Skills Icons (Communication, Digital Literacy, Teamwork, Critical Thinking, Problem Solving, Enterprising). Name files to match, e.g. `communication.svg`, `digital-literacy.svg`, etc.

The [ECC Communications Skills video](https://www.dropbox.com/scl/fi/vg3jxajsvfvn7gjd3mmdq/ECC_Communications-Skills.mp4?rlkey=s35m15zvio5z8vapmrfkal74r&e=1&st=mmp70l00&dl=0) is linked in the header when the logo area is present.
